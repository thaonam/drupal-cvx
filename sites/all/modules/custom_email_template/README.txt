
-- SUMMARY --

Custom email template provides templates for emails. This module 
provides templates for default drupal emails.

REQUIREMENTS
------------

This module requires the following modules:

 * Views (https://drupal.org/project/link)

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------

Custom email template provides templates for emails. 

Installations and Configuration
1. Download Module and enable it by an ordinary Drupal way.
2. Create some nodes for "Mail Template" content type.
3. For every node, you can upload header image and add menu links
 for headers and footers.
4. After creating nodes, go to admin/config/people/accounts
5. You can see under E-mails section, there is a dropdown list 
of templates and also checkboxes for include herder and footer links.

Once you have selected your desire template then email related to 
users contains header and footer links and also header image.
